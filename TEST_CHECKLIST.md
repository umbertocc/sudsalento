# Test checklist per sito Astro su GitHub Pages

1. Apri docs/index.html in un browser locale.
2. Verifica che la homepage si carichi correttamente.
3. Clicca sui link del menu e controlla che portino alle pagine:
   - /torre-vado
   - /torre-pali
   - /lido-marini
   - /gallipoli
   - /porto-cesareo
   - /marina-di-felloniche
   - ...
4. Controlla che le URL siano https://umbertocc.github.io/sudsalento/...
5. Verifica che le immagini e i CSS siano caricati.
6. Controlla i meta tag canonical e og:url nel codice sorgente delle pagine.
7. Se usi GitHub Pages:
   - Carica la cartella docs/ sul repository.
   - Vai su https://umbertocc.github.io/sudsalento/ e verifica che tutto funzioni.
8. Testa la sitemap.xml e robots.txt per i nuovi URL.

Se vuoi uno script automatico, chiedi pure!
